# Code of Conduct

We are committed to a respectful, harassment-free community.

## Our standards

- Be respectful and constructive.
- Assume good intent and give others room to learn.
- No harassment, discrimination, or personal attacks.
- No spam, doxxing, or disruptive behavior.

## Scope

This code of conduct applies to project spaces and community interactions.

## Enforcement

If you experience or witness unacceptable behavior, please contact the
maintainers by opening a GitHub issue. If the issue is sensitive, state that you
need a private channel and a maintainer will follow up.
